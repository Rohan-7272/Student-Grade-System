# Student Grade Tracker

## Overview

The **Student Grade Tracker** is a Java-based console application that allows users to manage student records and analyze academic performance. The program enables users to enter student names and marks, store the data using Java collections, and generate a summary report with useful statistics.

This project demonstrates the fundamentals of Java programming, including Object-Oriented Programming (OOP), classes and objects, encapsulation, collections, loops, methods, and user input handling.

## Features

* Add multiple student records
* Store student data using `ArrayList`
* Calculate the average marks
* Find the highest score
* Find the lowest score
* Display a summary of all students
* Console-based user interface

## Technologies Used

* Java
* Java Collections (`ArrayList`)
* Object-Oriented Programming (OOP)
* Scanner Class

## How It Works

1. The user enters the number of students.
2. The program accepts each student's name and marks.
3. Student records are stored in an `ArrayList`.
4. The application calculates:

   * Average marks
   * Highest marks
   * Lowest marks
5. A summary report of all students is displayed.
